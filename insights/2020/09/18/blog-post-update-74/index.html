microPlatform update 74

September 18, 2020

Linux microPlatform

Highlights

  • ATF updated to the final 2.3 release
  • Aktualizr-Lite updated to the revision 56f8357 (better app management and config reporting)
  • Container pre-loading removed from meta-lmp, now done via Factory CI
  • Curl updated to the 7.72.0 release
  • GCC updated to the 10.2.0 stable release
  • Glibc updated to the 2.32 release
  • Go updated to the 1.14.7 stable release
  • LSHW updated to revision 71cf63b
  • Libseccomp updated to the 2.5.0 release
  • Linux-firmware updated to the 20200817 snapshot
  • Linux-lmp updated to the v5.8.8 stable release
  • Linux-lmp-fslc-imx updated to v5.4.64 (based on NXP BSP 5.4.24 2.1.0)
  • Linux-lmp-lts updated to the v5.4.64 stable release
  • Meta-ARM (yocto upstream) was added to the default list of layers
  • Meta-toradex-nxp and meta-toradex-bsp-common layers were removed from LMP (due conflicts with other BSP layers)
  • Mfgtools enabled for apalis-imx6, apalis-imx8 and imx7ulpea-ucom
  • NXP BSP updated to the 5.4.24 2.1.0 release
  • New board support: imx7ulpea-ucom (from Embedded Artists)
  • Openvino updated to the 2020.4 release
  • Ostree updated to the 2020.5 release
  • Qemu updated to the 5.1.0 release
  • SQlite3 updated to the 3.33.0 release
  • Sudo updated to the 1.9.2 release
  • Signed and restricted boot support enabled for apalis-imx6, imx7ulpea-ucom and imx8mmevk.
  • Support for OPTEE-OS 3.10.0 and U-Boot 2020.04 (NXP BSP 5.4.24 2.1.0 based) added to meta-lmp (not yet enabled as default)
  • Support for the Toradex Linux BSP now provided by the linux-lmp-toradex-imx recipe
  • Systemd updated to the v246.2 release
  • Wireguard module updated to the 1.0.20200729 release
  • Wireguard tools updated to the 1.0.20200827 release
