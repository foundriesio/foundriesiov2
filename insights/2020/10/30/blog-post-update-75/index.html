FOUNDRIES.IO INSIGHTS

microPlatform update 75

October 30, 2020

Linux microPlatform

Highlights

  • New image: lmp-base-console-image (console image with support for OSTree + OTA + Docker-ce)
  • New kernel recipes: linux-lmp-rpi (Raspberry Pi downstream kernel), linux-lmp-xlnx (Xilinx downstream kernel)
  • New platform support: ARM Corstone700-MPS3 and UltraZed-EG-IOCC (based on the Xilinx Zynq® UltraScale+™ MPSoC)
  • Aktualizr-Lite updated to the revision f6f16a0 (based on Aktualizr 2020.8+fio)
  • Apalis-imx8 now fully compatible with the latest Toradex/NXP 5.4-based BSP
  • Containerd updated to the 1.4.x source branch
  • Docker support for RISC-V via docker-moby (docker 20.10.0-dev)
  • Docker-Compose updated to the 1.27.4 release
  • Go updated to the 1.15.2 stable release
  • LMP-Base: support for separated u-boot configuration fragment and improved support for several targets
  • Linux-lmp updated to the v5.8.16 stable release
  • Linux-lmp-fslc-imx updated to v5.4.72 (based on NXP BSP 5.4.24 2.1.0)
  • Linux-lmp-lts updated to the v5.4.72 stable release
  • Modemmanager updated to the 1.14.2 stable release
  • Openvino updated to the 2021.1 release
  • Ostree updated to the 2020.6 release
  • Raspberry Pi device-tree / overlays management moved to the standard config.txt / proprietary firmware (raspberrypi4 targets will require a clean flash due incompatible firmware / kernel updates by downstream)
  • Runc updated to the 1.0.0-rc92 tag
  • Support for Docker-app removed, replaced with Docker Compose Apps
  • Support for filtering and blocking image packages based on package license (e.g. GPLv3)
  • Support for importing NXP EdgeLock™ SE050 pre-installed keys and certificates into PKCS11/SKS/OP-TEE via the pkcs11-se050-import tool
  • Systemd-boot updated to 246.2
  • U-Boot-Fio 2020.10 now available (not yet default for all targets)
  • Weston updated to the 9.0.0 release
  • Wireguard module updated to the 1.0.20200908 release
  • Xilinx layers (meta-xilinx-bsp and meta-xilinx-tools) added to the default list of layers
