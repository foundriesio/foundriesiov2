HARDWARE

SUPPORTED DEVICES AND PLATFORMS

The microPlatforms are portable and architecture independent. There are three levels of hardware support:

  • Reference devices: thorough device-specific documentation and builds, extensive automated end-to-end testing, globally available, low-cost hardware
  • Community devices: device-specific documentation and builds, best-effort testing, and community support
  • Upstream-supported devices: reference documentation for using any hardware with upstream support

Foundries.io is working with SoC and MCU manufacturers to create a "microPlatform Certified" program.
