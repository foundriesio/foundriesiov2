HARDWARE

SUPPORTED DEVICES

The Linux microPlatform supports a wide range of hardware and processor architectures. There are two levels of hardware support.

  • Reference devices: We provide device-specific documentation and builds, extensive automated end-to-end testing, and enable these devices to be securable using hardware features
  • Community devices: We provide generic documentation and builds, best-effort testing, and community support

Note that you are not limited to our devices, as any hardware can be supported by FoundriesFactory. If you have any questions please contact hardware support and we can help.
