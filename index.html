Linux microPlatform

The OTA updatable Linux microPlatform (LmP) combines secure firmware with an IoT/Edge focused Linux distribution using OpenEmbedded/Yocto. You can include a Docker container application runtime as a build option to meet your use case demands. Foundries.io microPlatforms keep these technologies updated for your product development and deployment lifetime. By basing your development on the LmP, you can securely and remotely manage all your end product software, using standard tools.

LEARN MORE

Zephyr microPlatform

The Zephyr microPlatform (ZmP) combines the Zephyr RTOS with MCUboot secure bootloader and reference firmware applications. It offers a continuously tested, over-the-air updatable, cross architecture solution for microcontroller-based products. The ZmP supports standalone product development and deployment with any wireless technology, as well as seamless integration with the Linux microPlatform for gateway-based systems.

FoundriesFactory

The FoundriesFactory subscription enables you to customize the Linux microPlatform to your own hardware design, software IP and use case.
