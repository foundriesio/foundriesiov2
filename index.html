Linux microPlatform

The Linux microPlatform combines secure updatable firmware, a minimal Linux distribution built using OpenEmbedded/Yocto, and a container-based application runtime. We do the heavy lifting to keep the microPlatforms close to tip on all of these technologies, giving you access to a continuously tested and stabilized stream of updates you can base your Linux product development on. By basing your development on the Linux microPlatform, you can securely and remotely manage your product's software using standard tools.

LEARN MORE

Zephyr microPlatform

The Zephyr microPlatform combines the MCUboot secure bootloader, Zephyr RTOS, and reference firmware applications to provide a continuously tested, over-the-air updatable, cross architecture solution for microcontroller-based products. The Zephyr microPlatform works as a standalone basis for product development, and also seamlessly integrates with the Linux microPlatform for gateway/device systems.
